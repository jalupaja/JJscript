# lang
> a bs programming language, built for the purpose of building a programming language

## "Features"

### environments
- local environments using `{ ... }`
- local variables
- functions using `id = { ... }` or `id = {(ARGS) ... }`

### datatypes
- integer (only octal and hex (using 0x...))
- float
- string
- NONE
- bool ("true"|"false")
